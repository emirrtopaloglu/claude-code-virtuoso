# User Personas Directory

This directory stores detailed user persona documents created by the `/vision` skill.

## Purpose

User personas help ensure the product is designed for the right people with specific needs, goals, and pain points.

## Template

When creating a new persona, include:

```markdown
# [Persona Name]

## Demographics
- Age: [range]
- Profession: [role]
- Tech-savviness: [low/medium/high]
- Location: [region/context]

## Goals
- Primary goal: [main objective]
- Secondary goals: [other objectives]

## Pain Points
- [What problems they face]
- [Current workarounds]
- [ Frustrations with existing solutions]

## Behaviors
- [How they work]
- [Tools they use]
- [Decision-making process]

## Scenarios
- [When they use the product]
- [Context of use]
- [Success criteria]
```

## Usage

Personas are created during `/vision` product discovery and referenced throughout development to ensure user-centered design.
